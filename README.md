About
PAIR is a web-based talent and recruitment platform that connects graduates and professionals with recruiters through verified, project-based evidence of skills rather than traditional CV-based applications.
Built as part of the Capstone Integration Project at Edith Cowan University by Team TechnoWiz, 2026.
